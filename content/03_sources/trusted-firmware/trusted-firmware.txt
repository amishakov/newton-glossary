Title: Trusted Firmware

----

Date: 21 December 2024

----

Text:

An organization providing developers and OEMs with standardized secure software and code libraries for hardware based on the (link: /terms/arm text: ARM) processor architecture.

One of their projects is (link: /terms/mbed-tls text: Mbed TLS), an open source library that provides implementations of (link: /terms/tls text: TLS) and (link: /terms/ssl text: SSL) protocols for use in software development.
