The user manual for the (link: /terms/newton-backup-utility text: Newton Backup Utility), a consumer product developed by [Apple](https://www.apple.com/) for backing up data from Newton devices running earlier versions of (link: /terms/newton-os text: Newton OS) to Mac OS and Windows computers.
