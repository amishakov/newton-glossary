Title: Newton Backup Utility User’s Guide

----

Date: 30 July 2024

----

Text:

The user manual for the [Newton Backup Utility](/terms/newton-backup-utility), a consumer product developed by [Apple](https://www.apple.com/) for backing up data from Newton devices running earlier versions of [Newton OS](/terms/newton-os) to Mac OS and Windows computers.
