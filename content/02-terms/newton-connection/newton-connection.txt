Title: Newton Connection

----

Date:

----

Text:

The software component included in Apple’s [Newton Connection Kit](/terms/newton-connection-kit) which connects Newton devices to Mac OS and Windows computers, allowing users to exchange and [synchronize](/terms/synchronize) information with common desktop [applications](/terms/application).

Newton Connection is also an [application](/terms/application) for Mac OS X created by [Simon Bell](/sources/simon-bell) that essentially replaces the functionality found in Apple’s [Newton Connection Utilities](/terms/newton-connection-utilities) (NCU). Newton Connection allows the user to [backup](/terms/backup) and [restore](/terms/restore) information, install [packages](/terms/package), and transfer data to and from a Newton device. Also referred to as [NCX](/terms/ncx).

Backup files created by Newton Connection are identified by the `.nbku` file extension.
