Title: newtVNC

----

Date: 18 April 2017

----

Text:

A [VNC](/terms/vnc) client and server for [Newton OS](/terms/newton-os) developed by [Steve Weyer](/sources/steve-weyer). The newtVNC client software allows the user to view the screen and interacting with a remote computer system using a Newton device. The server software allows the user to view the screen and interacting with a Newton device from another computer.
