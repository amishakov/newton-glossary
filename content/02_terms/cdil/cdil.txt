Title: CDIL

----

Date: 24 May 2001

----

Text:

Communication Desktop Integration Layer

A code layer that provides a stream-based interface for exchanging data with Newton devices over a variety of communications services.

This code is part of the (link: /terms/desktop-integration-library text: Desktop Integration Library) ((link: /terms/dil text: DIL)) provided to developers by [Apple](https://www.apple.com/).
