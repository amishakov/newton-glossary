Title: Newton Connection

----

Date: 24 August 2016

----

Text:

The software component included in Apple’s (link: /terms/newton-connection-kit text: Newton Connection Kit) which connects Newton devices to Mac OS and Windows computers, allowing users to exchange and (link: /terms/synchronize text: synchronize) information with common desktop (link: /terms/application text: applications).

Newton Connection is also an (link: /terms/application text: application) for Mac OS X created by (link: /sources/simon-bell text: Simon Bell) that essentially replaces the functionality found in Apple’s (link: /terms/newton-connection-utilities text: Newton Connection Utilities) (NCU). Newton Connection allows the user to (link: /terms/backup text: backup) and (link: /terms/restore text: restore) information, install (link: /terms/package text: packages), and transfer data to and from a Newton device.

Also referred to as (link: /terms/ncx text: NCX).

Backup files created by Newton Connection are identified by the [.nbku](/terms/nbku) filename extension.
