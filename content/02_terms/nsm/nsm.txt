Title: NSM

----

Date: 16 January 2024

----

Text:

An implementation of a portable [NewtonScript](/terms/newtonscript) virtual machine developed by [Matej Horvat](/sources/matej-horvat). NSM does not interpret NewtonScript source code directly, instead it executes NewtonScript [bytecode](/terms/bytecode) contained within Newton [packages](/terms/package).
