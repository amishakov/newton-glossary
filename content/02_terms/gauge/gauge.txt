Title: Gauge

----

Date: 24 May 2001

----

Text:

A (link: /terms/user-interface text: user interface) element with a visual (link: /terms/marker text: marker) that acts as an (link: /terms/indicator text: indicator) of magnitude, position, probability, degree, or some other amount within a range of possible values. A user can only read the value indicated by a gauge, not manipulate the value like can be done using a (link: /terms/slider text: slider).

This type of indicator includes the (link: /terms/bar-gauge text: Bar Gauge), (link: /terms/barber-pole-gauge text: Barber Pole Gauge), and (link: /terms/progress-indicator text: Progress Indicator).
