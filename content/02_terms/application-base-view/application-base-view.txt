Title: Application Base View

----

Date: 24 May 2001

----

Text:

In (link: /terms/newtonscript text: NewtonScript), the topmost (link: /terms/parent-view text: parent view) in an (link: /terms/application text: application). The application base view typically encloses all other (link: /terms/view text: views) that make up the application.
