Title: Application Base View

----

Date: 24 May 2001

----

Text:

In [NewtonScript](/terms/newtonscript), the topmost [parent view](/terms/parent-view) in an [application](/terms/application). The application base view typically encloses all other [views](/terms/view) that make up the application.
