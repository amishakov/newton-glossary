Title: ROM Extension

----

Date:

----

Text:

Additional data pertaining to the [operating system](/terms/operating-system) that is external to the data in the [ROM](/terms/rom) itself, which contains the base [Newton OS](/terms/newton-os). Newton devices produced by [Apple](https://www.apple.com/) include the Apple ROM Extension ([REx](/terms/rex)) as the initial extension. The Apple REx contains most of the built-in Newton OS [applications](/terms/application). For example, on an [eMate](/terms/emate-300) the REx contains [Newton Works](/terms/newton-works). ROM extensions can also define various hardware features, such as system timing, memory specifications, and code.

[Clone](/terms/clone) or licensed Newton OS devices usually include the Apple REx along with a vendor-specific, custom ROM extension.

Up to four ROM extensions can be included on an [N2](/terms/n2) platform device.
