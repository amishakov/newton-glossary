Title: Show Button

----

Date: 27 December 2017

----

Text:

The [button](/terms/button) found in an [application](/terms/application) that allows the user to select different views of the same data. When [tapped](/terms/tapped), the button displays a [picker](/terms/picker) listing alternative views available for the current application.
