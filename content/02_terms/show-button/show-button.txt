Title: Show Button

----

Date: 27 December 2017

----

Text:

The (link: /terms/button text: button) found in an (link: /terms/application text: application) that allows the user to select different views of the same data. When (link: /terms/tapped text: tapped), the button displays a (link: /terms/picker text: picker) listing alternative views available for the current application.
