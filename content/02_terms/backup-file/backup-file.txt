Title: Backup File

----

Date: 28 April 2017

----

Text:

A file residing on a (link: /terms/storage-card text: storage card), (link: /terms/desktop-computer text: desktop computer), or (link: /terms/laptop-computer text: laptop computer) that contains a copy of the information from a Newton device.

A backup file created by (link: /terms/newton-backup-utility text: Newton Backup Utility) or (link: /terms/newton-connection-utilities text: Newton Connection Utilities) of a (link: /terms/storage-card text: storage card) that has not been named in the (link: /terms/card-slip text: card slip) is known generically as a (link: /terms/card-file text: card file).
