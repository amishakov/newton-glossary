Title: Script

----

Date: 24 August 2016

----

Text:

A program written in a programming language (such as (link: /terms/newtonscript text: NewtonScript)) that generally uses untyped (link: /terms/variable text: variables) — that is, variables that can hold values of any data type — and seldom supports complex data structures.

A script can also be a series of simple commands used to configure a peripheral such as a (link: /terms/modem text: modem).
