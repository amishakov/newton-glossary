Title: Message Module

----

Date: 05 November 2017

----

Text:

An [extension](/terms/extension) installed alongside the [Newton Messaging Enabler](/terms/newton-messaging-enabler) which provides communication level functionality for a specific messaging device, such as the [Newton Messaging Card](/terms/newton-messaging-card), Socket [PageCard](/terms/pagecard), or Motorola [Tango](/terms/tango) pager.
