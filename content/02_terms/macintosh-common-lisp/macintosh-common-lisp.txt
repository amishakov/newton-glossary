Title: Macintosh Common Lisp

----

Date: 29 June 2017

----

Text:

An implementation of the (link: /terms/lisp text: Lisp) programming language for Mac OS computers, it was also a development environment that provided tight integration with the Mac OS toolbox.

Also referred to as (link: /terms/mcl text: MCL).
