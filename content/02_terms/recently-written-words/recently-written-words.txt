Title: Recently Written Words

----

Date: 14 October 2017

----

Text:

A (link: /terms/slip text: slip) which displays a list of words from the (link: /terms/auto-add-dictionary text: auto add dictionary) when the user opens the (link: /terms/personal-word-list text: Personal Word List). Words in this list are not contained in the built-in (link: /terms/system-dictionary text: system dictionary) and have been automatically added to the (link: /terms/user-dictionary text: user dictionary).

The Recently Written Words slip prompts the user to indicate whether each of the words it displays should remain in the user dictionary.
