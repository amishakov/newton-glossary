Title: System Memory

----

Date: 24 May 2001

----

Text:

The portion of memory used by a Newton device in which programs run and any intermediate data created by those programs is stored. System memory is often confused with (link: /terms/system-ram text: system RAM), which is the amount of physical (link: /terms/ram text: RAM) installed in a Newton device.

Also referred to as (link: /terms/heap-space text: Heap Space).
