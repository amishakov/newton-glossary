Title: Base Classroom State

----

Date: 29 December 2023

----

Text:

A group of settings specified using the (link: /terms/teacher-setup text: Teacher Setup) application that are common to all (link: /terms/emate-300 text: eMate 300) devices in a classroom. These settings can include a default printer, whether spell-checking is enabled, whether multiple users can log into an eMate, as well as which (link: /terms/application text: applications) users are able to access. Once you set a base classroom state on a single eMate, you can transfer those settings to other eMate devices using a (link: /terms/storage-card text: storage card).
