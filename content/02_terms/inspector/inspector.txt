Title: Inspector

----

Date: 09 October 2017

----

Text:

A (link: /terms/debugging text: debugging) window in the (link: /terms/newton-toolkit text: Newton Toolkit) development environment that allows a user to browse the (link: /terms/object text: object) storage system and execute (link: /terms/newtonscript text: NewtonScript) code on a Newton device.

The Inspector communicates with the (link: /terms/toolkit text: Toolkit) application installed on a Newton device connected to a (link: /terms/desktop-computer text: desktop computer) or (link: /terms/laptop-computer text: laptop computer) via a serial cable or (link: /terms/appletalk text: AppleTalk) network.
