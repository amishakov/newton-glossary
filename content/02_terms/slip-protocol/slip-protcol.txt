Title: SLIP

----

Date: 20-Mar-2024

----

Text:

Serial Line Internet Protocol

Software that allows the [IP](/terms/ip) network layer, commonly used over [ethernet](/terms/ethernet), to be used over a serial connection such as a [modem](/terms/modem).

SLIP was superceded by [PPP](/terms/ppp).

For the user interface element, see [Slip](/terms/slip).
