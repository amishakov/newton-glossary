Title: Endpoint Option

----

Date: 24 May 2001

----

Text:

In (link: /terms/newtonscript text: NewtonScript), an (link: /terms/endpoint text: endpoint) option is specified in an (link: /terms/option-frame text: option frame) passed in an (link: /terms/array text: array) as an argument to one of the endpoint (link: /terms/method text: methods). Endpoint options select the communication tool to use, control its configuration and operation, and (link: /terms/return text: return) result code information from each method call.
