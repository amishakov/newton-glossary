Title: Address

----

Date: 31 May 2001

----

Text:

A binary value specifying where in the memory a chunk of data or code exists.

In a (link: /terms/spreadsheet text: spreadsheet) document, the address of a (link: /terms/cell text: cell) corresponds to the column and the row where it is located.
