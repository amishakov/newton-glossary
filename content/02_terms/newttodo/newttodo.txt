Title: newtToDo

----

Date: 09 November 2017

----

Text:

Software developed by (link: /sources/adam-tow text: Adam Tow) that allows the user to efficiently manage tasks using a Newton device. Features include reminders with (link: /terms/alarm text: alarms), repeating tasks, additional priority levels, flexible options for sorting and filing, a customizable (link: /terms/user-interface text: user interface), and the ability to automatically import (link: /terms/to-do-list text: To Do List) items from the (link: /terms/date-book text: Date Book) application.
