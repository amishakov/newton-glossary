Title: Clock

----

Date: 10 May 2024

----

Text:

The built-in (link: /terms/application text: application) that displays the current time using an analog style clock face. The application allows the user to change the time and date, set daily alarms, and set minute timers. The clock also can be accessed by (link: /terms/tapping text: tapping) the time and date that appears in the upper left corner of the (link: /terms/notepad text: Notepad) application.
