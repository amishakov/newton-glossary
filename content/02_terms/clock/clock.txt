Title: Clock

----

Date: 10 May 2024

----

Text:

The built-in [application](/terms/application) that displays the current time using an analog style clock face. The application allows the user to change the time and date, set daily alarms, and set minute timers. The clock also can be accessed by [tapping](/terms/tapping) the time and date that appears in the upper left corner of the [Notepad](/terms/notepad) application.
