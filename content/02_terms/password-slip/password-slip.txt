Title: Password Slip

----

Date: 13 June 2025

----

Text:

A (link: /terms/slip text: slip) that allows the user to create a new (link: /terms/password text: password) or enter an existing password without echoing the password in plain text. Typed (link: /terms/character text: characters) appear as bullets in the (link: /terms/input-line text: input lines).

(image: /assets/images/password-slip-with-keyboard-2x.gif alt: Screenshot of the Password slip with a typewriter keyboard width: 299)

When displayed on an (link: /terms/emate-300 text: eMate 300) or when a physical keyboard such as the (link: /terms/newton-keyboard text: Newton keyboard) is connected to a Newton device, the password slip does not include a (link: /terms/typewriter text: typewriter) keyboard.
