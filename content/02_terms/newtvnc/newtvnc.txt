Title: newtVNC

----

Date: 18 April 2017

----

Text:

A (link: /terms/vnc text: VNC) client and server for (link: /terms/newton-os text: Newton OS) developed by (link: /sources/steve-weyer text: Steve Weyer). The newtVNC client software allows the user to view the screen and interacting with a remote computer system using a Newton device. The server software allows the user to view the screen and interacting with a Newton device from another computer.
