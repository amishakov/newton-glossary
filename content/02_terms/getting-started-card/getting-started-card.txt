Title: Getting Started Card

----

Date: 08 August 2025

----

Text:

A (link: /terms/pcmcia text: PCMCIA) card that shipped with the (link: /terms/original-messagepad text: Original MessagePad) and (link: /terms/messagepad-100 text: MessagePad 100) along with a printed (link: /sources/messagepad-setup-guide text: setup guide).

The card contains three (link: /terms/application text: applications) to help the user learn about the Newton device and how to use various features of (link: /terms/newton-os text: Newton OS).

(link: /terms/instructor text: Instructor) provided tips and techniques for using the (link: /terms/handwriting-recognition text: handwriting recognition).

(link: /terms/tour text: Tour) walked through various capabilities of the device.

(link: /terms/game text: Game) encouraged the user to practice their handwriting through an arcade style game, gathering information to help improve the accuracy of the handwriting recognition.
