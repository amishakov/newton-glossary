Title: User Storage

----

Date: 24 May 2001

----

Text:

The portion of memory used for permanent user data and installed (link: /terms/package text: packages). User storage is analogous to disk storage on a (link: /terms/desktop-computer text: desktop computer) or (link: /terms/laptop-computer text: laptop).
