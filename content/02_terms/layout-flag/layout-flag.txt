Title: Layout Flag

----

Date:

----

Text:

A [Newton Book Maker](/terms/newton-book-maker) setting value used to modify a [layout command](/terms/layout-command). Setting this [flag](/terms/flag) affects all pages using a particular [layout](/terms/layout).
