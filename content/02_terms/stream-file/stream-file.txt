Title: Stream File

----

Date:

----

Text:

A file encoded in Newton Streamed Object Format ([NSOF](/terms/nsof)). You can use [Newton Toolkit](/terms/newton-toolkit) to build stream files, which can be incorporated into [projects](/terms/project).
