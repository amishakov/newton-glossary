Title: Stream File

----

Date: 24 May 2001

----

Text:

A file encoded in Newton Streamed Object Format ((link: /terms/nsof text: NSOF)). You can use (link: /terms/newton-toolkit text: Newton Toolkit) to build stream files, which can be incorporated into (link: /terms/project text: projects).
