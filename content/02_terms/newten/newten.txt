Title: NewTen

----

Date:

----

Text:

An [application](/terms/application) for Mac OS X created by [Steven Frank](/sources/steven-frank) that allows the user to install [packages](/terms/package) on a Newton device or download a copy of the [ROM](/terms/rom) over a serial connection. NewTen is based on [UnixNPI](/terms/unixnpi), a command line [package installer](/terms/package-installer) for Unix [operating systems](/terms/operating-system).
