Title: Info Button

----

Date: 06 December 2017

----

Text:

A small square (link: /terms/button text: button) containing an (link: /terms/icon text: icon) representing an “I” (link: /terms/character text: character) and located in the (link: /terms/status-bar text: status bar) of an (link: /terms/application text: application). When (link: /terms/tapped text: tapped), the button displays a (link: /terms/picker text: picker) which gives the user access to (link: /terms/preferences text: preferences), onscreen help, or general information about the application.

The info button in the (link: /terms/extras-drawer text: Extras Drawer) lists (link: /terms/memory-info text: Memory Info) as an option in the picker. When selected, this displays the (link: /terms/memory-info-slip text: Memory Info Slip).

Also known as the (link: /terms/information-button text: Information Button) in earlier versions of (link: /terms/newton-os text: Newton OS).
