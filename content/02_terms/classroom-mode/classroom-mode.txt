Title: Classroom Mode

----

Date: 29 December 2023

----

Text:

A mode of operation on an (link: /terms/emate-300 text: eMate 300) typically used in a classroom or multiuser situation. This mode is specified using the (link: /terms/teacher-setup text: Teacher Setup) application and provides a simplified (link: /terms/newton-user-interface text: Newton user interface), limiting which features and (link: /terms/application text: applications) students can access.

The alternative to Classroom Mode is (link: /terms/full-newton-system text: Full Newton System) mode.
