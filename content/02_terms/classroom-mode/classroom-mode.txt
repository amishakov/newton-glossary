Title: Classroom Mode

----

Date: 29 December 2023

----

Text:

A mode of operation on an (link: /terms/emate-300 text: eMate 300) typically used in a classroom or multiuser situation. Classroom Mode is specified using the (link: /terms/teacher-setup text: Teacher Setup) application and provides a simplified Newton interface, limiting which features and (link: /terms/application text: applications) students can access.
