Title: Classroom Mode

----

Date: 29 December 2023

----

Text:

A mode of operation on an [eMate 300](/terms/emate-300) typically used in a classroom or multiuser situation. Classroom Mode is specified using the [Teacher Setup](/terms/teacher-setup) application and provides a simplified Newton interface, limiting which features and [applications](/terms/application) students may access.
