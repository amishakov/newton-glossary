Title: FDIL

----

Date: 24 May 2001

----

Text:

Frame Desktop Integration Library

A collection of code written in the [C](https://en.wikipedia.org/wiki/C_(programming_language)) programming language responsible for the management of (link: /terms/newtonscript text: NewtonScript) compatible objects, which supercedes the High Level Frame Desktop Integration Library or (link: /terms/hlfdil text: HFDIL).

This code is part of the (link: /terms/desktop-integration-libraries text: Desktop Integration Libraries) ((link: /terms/dil text: DIL)) provided to developers by [Apple](https://www.apple.com/).
