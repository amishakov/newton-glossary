Title: Keystroke Events

----

Date: 12 August 2024

----

Text:

[Events](/terms/event) generated by [Newton OS](/terms/newton-os) when the user interacts with a [keyboard](/terms/keyboard). Keystroke events can include the `key-down` event when a key is pressed, the `key-up` event when a key is released, and `key-repeat` event when a key is held down for a specific length of time.
