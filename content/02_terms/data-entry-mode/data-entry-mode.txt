Title: Data Entry Mode

----

Date: 24 February 2025

----

Text:

The ability to (link: /terms/enter text: enter) or edit information in a (link: /terms/spreadsheet text: spreadsheet).

By (link: /terms/tapping text: tapping) on a spreadsheet (link: /terms/cell text: cell), the cell is (link: /terms/highlight text: highlighted) and the user is able to enter information into that cell. On an (link: /terms/emate-300 text: eMate 300), pressing the (link: /terms/return-key text: Return key), (link: /terms/tab-key text: Tab key), or (link: /terms/enter-key text: Enter key) on the keyboard will take the user out of data entry mode. When not in data entry mode, pressing an arrow key, the Return key, or the Tab key will move the highlight to another cell.
