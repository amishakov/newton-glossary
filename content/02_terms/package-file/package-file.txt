Title: Package File

----

Date: 24 May 2001

----

Text:

A file that contains software, such as an (link: /terms/application text: application) program or (link: /terms/extension text: extension), which can be transferred to a Newton device using a (link: /terms/package-installer text: package installer). Identified by the [.pkg](/terms/pkg) or [.newtonpkg](/terms/newtonpkg) filename extension.
