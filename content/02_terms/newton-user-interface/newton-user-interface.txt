Title: Newton User Interface

----

Date: 24 May 2001

----

Text:

The standard rules and conventions for interacting with Newton devices, as set out in the (link: /sources/newton-user-interface-guidelines text: Newton User Interface Guidelines) documentation. The (link: /terms/user-interface text: user interface) provides a consistent means of interacting with all Newton devices and any (link: /terms/application text: applications) designed to run on those devices.

Also referred to as the (link: /terms/nui text: NUI).
