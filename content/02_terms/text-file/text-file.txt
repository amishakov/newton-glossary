Title: Text File

----

Date: 24 May 2001

----

Text:

A file containing text-based data that is compiled by [Newton Toolkit](/terms/newton-toolkit) during a [project build](/terms/project-build). Usually identified by the [.txt](/terms/txt) filename extension.
