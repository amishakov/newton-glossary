Title: Routing Format

----

Date: 24 May 2001

----

Text:

In (link: /terms/newtonscript text: NewtonScript), a (link: /terms/frame text: frame) that describes how to format an (link: /terms/object text: object) that is to be sent (or routed). Examples include print routing formats which describe how to visually format data and frame routing formats which describe the internal structure of a frame.
