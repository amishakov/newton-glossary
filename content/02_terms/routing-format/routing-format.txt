Title: Routing Format

----

Date:

----

Text:

In [NewtonScript](/terms/newtonscript), a [frame](/terms/frame) that describes how to format an [object](/terms/object) that is to be sent (or routed). Examples include print routing formats which describe how to visually format data and frame routing formats which describe the internal structure of a frame.
