Title: New Button

----

Date: 02 December 2017

----

Text:

The [button](/terms/button) found in an [application](/terms/application) that allows the user to create a new item. When [tapped](/terms/tapped), the button displays a [picker](/terms/picker) listing the various types of items available for the current application.

For example, creating a new [note](/terms/note) or [checklist](/terms/checklist) in the [Notepad](/terms/notepad), a new [card](/terms/card) for a person or group in the [Name File](/terms/name-file), or a new [meeting](/terms/meeting) in the [Date Book](/terms/date-book).
