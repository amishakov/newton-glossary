Title: New Button

----

Date: 02 December 2017

----

Text:

The (link: /terms/button text: button) found in an (link: /terms/application text: application) that allows the user to create a new item. When (link: /terms/tapped text: tapped), the button displays a (link: /terms/picker text: picker) listing the various types of items available for the current application.

For example, creating a new (link: /terms/note text: note) or (link: /terms/checklist text: checklist) in the (link: /terms/notepad text: Notepad), a new (link: /terms/card text: card) for a person or group in the (link: /terms/name-file text: Name File), or a new (link: /terms/meeting text: meeting) in the (link: /terms/date-book text: Date Book).
