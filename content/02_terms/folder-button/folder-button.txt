Title: Folder Button

----

Date: 20 August 2017

----

Text:

A [button](/terms/button) located in the [Notepad](/terms/note), [Name File](/terms/name-file), [Extras Drawer](/terms/extras-drawer), and other [applications](/terms/application) that allows the user to file items into [folders](/terms/folder). When [tapped](/terms/tapped), it displays a [folder slip](/terms/folder-slip) listing the folders where selected items can be filed.

Also referred to as a [Filing Button](/terms/filing-button).
