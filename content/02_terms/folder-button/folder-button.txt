Title: Folder Button

----

Date: 20 August 2017

----

Text:

A (link: /terms/button text: button) located in the (link: /terms/note text: Notepad), (link: /terms/name-file text: Name File), (link: /terms/extras-drawer text: Extras Drawer), and other (link: /terms/application text: applications) that allows the user to file items into (link: /terms/folder text: folders). When (link: /terms/tapped text: tapped), it displays a (link: /terms/folder-slip text: folder slip) listing the folders where selected items can be filed.

Also referred to as a (link: /terms/filing-button text: Filing Button).
