Text:

High Level Frame Desktop Integration Layer

A code layer responsible for the management of (link: /terms/newtonscript text: NewtonScript) compatible objects, superceded by the Frame Desktop Integration Layer or (link: /terms/fdil text: FDIL).

This code is part of the (link: /terms/desktop-integration-library text: Desktop Integration Library) ((link: /terms/dil text: DIL)) provided to developers by [Apple](https://www.apple.com/).
