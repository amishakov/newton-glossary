Title: Auto Add Dictionary

----

Date: 14 October 2017

----

Text:

A list of words that have been automatically added to the (link: /terms/user-dictionary text: user dictionary). If the auto add dictionary is not empty, the (link: /terms/recently-written-words text: Recently Written Words) slip displays its contents when the user opens the (link: /terms/personal-word-list text: Personal Word List) slip.

The auto add dictionary is limited to 100 words.
