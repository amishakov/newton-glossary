Title: MessagePad 150

----

Date: 18 August 2024

----

Text:

A rumoured successor to the [MessagePad 120](/terms/messagepad-120), reportly smaller than existing Newton devices and including an optional [backlit](/terms/backlit) display.

The [Makido](/sources/makido) website lists the code name for the MessagePad 150 as [Q](/terms/q). This confusion is likely due to the fact that the code names for the next two [Newton OS](newton-os) devices released by Apple were very similar. The code name for the [MessagePad 130](/terms/messagepad-130) was [Quark](/terms/quark) and the code name for the [MessagePad 2000](/terms/messagepad-2000) was Q.

Also referred to as the [Newton 150](/terms/newton-150) in [The New York Times](/sources/the-new-york-times).
