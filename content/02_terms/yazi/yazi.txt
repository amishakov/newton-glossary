Yet Another Z Interpreter

A text adventure engine developed by [Scrawl Software](/sources/scrawl-software) which reads [packages](/terms/package) containing source data, allowing you to play classic games such as Zork, Planetfall, and Enchanter on a Newton device.

The YAZI [application](/terms/application) is a [bytecode](/terms/bytecode) interpreter for games written using the Zork Implementation Language (ZIL) which shares similarities with the [LISP](/terms/lisp) programming language.
