Title: Full Newton System

----

Date: 12 May 2025

----

Text:

A mode of operation on an (link: /terms/emate-300 text: eMate 300) that makes all features and (link: /terms/application text: applications) available to a single user. This mode is specified using the (link: /terms/teacher-setup text: Teacher Setup) application.

The alternative to Full Newton System mode is (link: /terms/classroom-mode text: Classroom Mode) which is typically used where multiple users are accessing the same device.
