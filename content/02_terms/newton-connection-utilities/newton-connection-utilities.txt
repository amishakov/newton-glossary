Title: Newton Connection Utilities

----

Date: 24 May 2001

----

Text:

Software developed by [Apple](https://www.apple.com/) that allows the user to exchange and (link: /terms/synchronize text: synchronize) information between a Newton device and common desktop (link: /terms/application text: applications). This software also offers support for installing (link: /terms/package text: packages) and using a physical keyboard as an input peripheral. The Newton Connection Utilities is for use with (link: /terms/newton-os text: Newton OS) 2.0 and later. The equivalent software package for earlier versions of Newton OS is (link: /terms/newton-connection text: Newton Connection).

The code name for Newton Connection Utilities was (link: /terms/gyro text: Gyro).

Also referred to as (link: /terms/ncu text: NCU).
